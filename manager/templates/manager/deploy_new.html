How it works
  1. Select a GitHub account with a PAT that has repo + workflow scopes.
  2. Enter your VPLink key — the short code from your vplink.in URL.
  3. Choose Private or Public repo visibility.
  4. Optionally link proxy credentials for IP rotation.
  5. Click Deploy — the system creates a repo, pushes the automation code, sets secrets, and triggers the first run.
  6. Monitor everything from the Dashboard — view counts, destination URLs, and run status.
